# AI Personality Detection Project

This project is a simple Artificial Intelligence system that predicts personality traits based on social media text or captions.

The model analyzes text input and classifies whether the user shows introvert or extrovert tendencies.

## Technologies Used
Python
Machine Learning
Natural Language Processing
Scikit-learn
Pandas

## How the Project Works
1. Dataset of captions is collected.
2. Text is converted into numerical format using TF-IDF.
3. Logistic Regression model is trained.
4. User enters a caption.
5. Model predicts personality type.

## How to Run the Project

Install dependencies:

pip install -r requirements.txt

Run the program:

python analyzer.py

Type captions and the model will predict personality.

Type "exit" to stop the program.
